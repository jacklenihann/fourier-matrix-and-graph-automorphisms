# Fourier Matrix and Graph Automorphisms

This repository contains SageMath code used in the bachelor thesis
Graphs Corresponding to Complex Hadamard Matrices and Their Symmetries.

The code computes, for a prime number p:

- the number of vertices of the graph Gamma_F_p,
- the number of automorphisms of the Fourier matrix F_p,
- the number of automorphisms of the graph Gamma_F_p preserving the row and column parts,
- the number of all automorphisms of the graph Gamma_F_p,
- the ratio |Aut_RC(Gamma_F_p)| / |Aut(F_p)|.
